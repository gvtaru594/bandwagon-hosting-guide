## 搬瓦工建站环境怎么搭？三种方案对比

买了 VPS 之后，需要在上面搭建 Web 服务环境。目前主流有三种方案：

### 方案一：LNMP 一键包（推荐新手+老手都用）

LNMP = Linux + NGINX + Mysql + PHP。NGINX 比 Apache 占用内存低，1GB 小内存机器首选。

安装流程：

1. SSH 登录搬瓦工 VPS（KiwiVM 控制面板可以直接看到 SSH 端口和 root 密码）
2. 下载并运行 LNMP 一键安装包
3. 安装过程中选择 MySQL 版本（1GB 内存选 5.5 或 5.6，2GB 及以上可以选 5.7）
4. 安装完成后，用 `lnmp vhost add` 命令添加网站域名
5. 配置伪静态规则（WordPress 选 WordPress 模式）
6. 上传 WordPress 程序文件到对应目录（`/home/wwwroot/你的域名/`）
7. 浏览器访问域名，完成 WordPress 安装向导

安装完整耗时大概 20-40 分钟，主要是编译时间。

### 方案二：宝塔面板（图形界面，更直观）

宝塔有 Web 界面，添加网站、管理数据库、SSL 证书都在界面上点点就行。对命令行不熟悉的朋友这个更友好。

安装一行命令搞定，然后登录面板 → 推荐安装 LNMP 套件 → 添加站点 → 上传程序。

要注意的是，宝塔本身也占内存，1GB 的机器会比较紧。

### 方案三：LAMP（Apache + PHP + MySQL）

Apache 比 NGINX 吃内存，除非有特殊需要，内存小的机器不推荐。

**简单总结**：小内存机器 → LNMP；不懂命令行 → 宝塔；内存够用、需要特定 Apache 功能 → LAMP。

## FAQ

**Q：搬瓦工建站速度慢怎么办？**

大概率是机房选的不合适，或者用的是普通 CN2 GT 线路。第一步先确认你买的套餐是不是 CN2 GIA-E 系列，如果是，在 KiwiVM 里尝试迁移到 DC9 或日本软银机房，重新 ping 一下看延迟是否改善。另外，站点本身没有开启缓存插件也会影响速度，WordPress 建议装 WP Super Cache 或 W3 Total Cache。

**Q：搬瓦工 1GB 内存够建站用吗？**

跑 LNMP + WordPress，1GB 够用。但宝塔面板会额外占内存，1GB 宝塔环境下跑 MySQL 会比较紧张。如果预算有限就用 LNMP，有余量的话直接上 2GB 版本。

**Q：搬瓦工支持支付宝吗？**

支持支付宝和 PayPal。国内用户用支付宝付款很方便，结算时选对货币即可。

**Q：CN2 GIA-E 和 KVM 套餐有什么区别，建站用哪个？**

线路不同。KVM 套餐走普通线路或 CN2 GT，CN2 GIA-E 走三网 CN2 GIA 企业级高速线路。面向国内用户建站，CN2 GIA-E 的访问速度和稳定性明显更好，值得多花这部分钱。
